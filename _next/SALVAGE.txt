── KEY INVARIANTS TO PRESERVE ────────────────────────────────────────────────

Grammar dict schema (new):
{
  "seed": {"type": "tetrahedron|cube|icosahedron", "radius": float},
  "iterations": [
    {
      "operation":    "subtract|add|intersect",
      "primitive":    "tetrahedron|cube|icosahedron|sphere",
      "fd_u":         float ∈ [0,1],
      "fd_v":         float ∈ [0,1],
      "scale_factor": float,
      "smooth_radius": float,
      "apply_to":     "all|new"
    }
  ]
}
symmetry_group is derived: SEED_TO_GROUP[seed.type]

Fitness contract:
  all sub-scores ∈ [0,1]
  hard gates return fitness=0.0
  weights sum to 1.0
  symmetry_preservation either removed or fixed at 1.0

Genetic contract:
  crossover: child inherits seed from one parent (determines G);
             operations borrowed freely from either parent;
             fd_u/fd_v of borrowed ops re-sampled within new FD if G changed
  mutation:  fd_u/fd_v perturbed with Gaussian σ=0.15, clamped [0,1]
  diversity: diverse_population() seeds equal thirds across the 3 types
